1. Go to https://tinypng.com/
2. Upload `gadget_zone_logo.png`
3. Download compressed version
4. Replace original file
5. **Expected result**: ~80KB (78% smaller)

1. Go to https://squoosh.app/
2. Upload the logo file
3. Select "WebP" output format
4. Set quality to 85%
5. Download both WebP and PNG versions
6. **Expected result**: ~25KB WebP (93% smaller)
